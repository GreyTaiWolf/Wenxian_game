# 物品装备与技能图鉴

本文档用于集中登记当前版本的物品、装备、词条和技能信息，方便后续策划扩展时查名字、用途、属性和获取方法。

实际运行数据仍以 `src/data/*.ts` 与 `src/game/*.ts` 为准。新增或调整物品、装备、词条、技能、掉落、商店、任务、突破消耗或战斗公式时，需要同步更新本文档，并按影响范围继续同步 `docs/GAME_DESIGN.md` 或 `docs/BALANCE.md`。

## 维护口径

- 物品配置源：`src/data/items.ts`
- 装备生成源：`src/game/generateEquipment.ts`、`src/data/weaponAttackRanges.ts`、`src/data/slotMainStatRules.ts`
- 装备品级源：`src/data/qualityGrades.ts`
- 装备词条源：`src/data/affixPools.ts`、`src/data/affixGenerateRules.ts`
- 技能配置源：`src/data/skills.ts`
- 获取来源源：`src/game/state.ts`、`src/data/world.ts`、`src/data/enemies.ts`、`src/data/progression.ts`、`src/data/cave.ts`

## 通用字段

| 字段 | 说明 |
| --- | --- |
| ID | 代码引用使用英文 `lower_snake_case`，不得随意改名。 |
| 名称 | 玩家可见中文名。装备显示名运行时会变为 `{品级前缀}·{名称}`。 |
| 类型 | `equipment` 装备、`pill` 丹药、`material` 材料、`quest` 任务物品。 |
| 器类阶位 | 物品所属大境界：凡阶、炼气阶、筑基阶等，决定装备主属性和内容投放层级。 |
| 当前品级 | 凡品、良品、精品、灵品、玄品、地品、天品、仙品、神品，决定稀有度、价格倍率、词条数量和特效。 |
| 用途 | 当前已实装用途优先；未接入效果的内容标为预留。 |
| 属性 | 装备记录槽位和生成规则，丹药记录战斗回复，材料记录消耗场景。 |
| 获取方法 | 只记录当前代码中能拿到的来源；测试注入、预留和敌人专用会明确标注。 |

## 装备总规则

装备采用实例制。同名装备进入背包时会生成独立 `EquipmentInstance`，并固化 `displayName / realmTier / quality / slot / mainStats / affixes / powerBonus`。当前实例属性由装备生成器计算，不直接使用 `items.ts` 中旧静态 `baseBonuses` 作为最终战斗属性。

装备主属性先读取同阶同品级武器攻击范围 `W`，再按部位换算。副词条按境界基础范围、品级系数、槽位限制和词条权重随机生成。高一大境界装备可试穿，但处于封印状态：主属性只生效 60%，玄品及以上特殊词条封印。

| 槽位 ID | 中文槽位 | 主属性规则 | 定位 |
| --- | --- | --- | --- |
| `weapon` | 武器 | 攻击 = W x 1.00-1.00 | 主要输出来源 |
| `robe` | 法袍 | 防御 = W x 0.45-0.60，气血 = W x 4.00-7.00 | 生存 |
| `helmet` | 头冠 | 气血 = W x 3.00-5.00，防御 = W x 0.25-0.40 | 生存 |
| `wrist` | 护腕 | 攻击 = W x 0.15-0.25，防御 = W x 0.20-0.35 | 攻防过渡 |
| `boots` | 鞋履 | 速度读取专表，闪避读取专表 | 速度与闪避 |
| `ring` | 戒指 | 暴击读取专表，攻击 = W x 0.12-0.22 | 暴击与攻击 |
| `amulet` | 护符 | 灵力 = W x 2.50-4.50，闪避读取专表 | 资源与闪避 |
| `artifact` | 法宝 | 攻击 = W x 0.60-1.10，元婴后可生成神识 | 构筑、主动、领域和法则 |

## 技能字段

| 字段 | 说明 |
| --- | --- |
| 类型 | `cultivator` 为修士技能，`beast` 为灵宠或妖兽技能。 |
| 使用者 | `player` 玩家、`companion` 同伴、`enemyCultivator` 敌对修士、`pet` 灵宠、`beast` 妖兽。 |
| 目标 | `enemySingle` 敌方单体、`enemyAll` 敌方全体、`allySingle` 友方单体。 |
| 命中 | `fullDodge` 吃完整闪避，`halfDodge` 吃 50% 闪避，`noDodge` 不可闪避。 |
| 效果 | `damage` 伤害，`heal` 治疗，`reduceDamage` 减伤守护。 |
| AI 权重 | 敌我自动行动选择时的基础权重，越高越常用。 |
