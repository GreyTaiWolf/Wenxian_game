# 游戏设计文档

本文档记录修仙文字 RPG 的长期设计方向。任何新增玩法、内容、地图、系统、解锁或叙事结构时，都必须同步更新本文档。

## 核心体验

玩家从青云城外醒来，以散修身份开始修仙。早期目标是修炼聚气、积累资源、探索中州、加入青云宗、获得灵宠或同伴，并最终突破到筑基阶段。游戏以文字反馈、轻量地图和回合制战斗推进。

当前体验支柱：

- 修炼：通过聚气获得修为，满足境界要求后尝试突破。
- 突破：消耗灵石和材料，按成功率提升境界，并解锁新系统。
- 背包：记录材料、丹药、任务物品和装备展示。
- 历练：通过大世界地图进入州域、地点和场景，触发对话、采集、战斗、任务、入宗和招募。
- 战斗：玩家和最多两名队友对抗妖兽或敌对修士，采用速度排序和回合行动。
- 宗门：加入青云宗后获得身份、贡献、声望和任务联系。
- 洞府：当前开放入口，未来承载闭关、聚灵阵和离线收益。
- 存档：本地三槽位存档，支持设置、继续、新建和删除。

## 当前世界内容

### 中州

中州是早期修行和新手内容中心。

- 青云城：主城，包含城主府、城东大道、城西坊市、城南民居和城北任务榜。
- 黑风山：野外战斗区，包含山狼、黑风妖修和妖兽骨产出。
- 灵药谷：采集和战斗区，提供凝气草、灵草，并触发青羽狐事件。
- 古修洞府：秘境区，提供洞府守卫战斗和筑基丹、青木灵液等机缘。
- 落霞镇：任务叙事地点，用于送信任务和商路传闻。

### 南疆

南疆是当前开放的第二个州域，主题为巫妖盟、木灵宗、草木灵药、瘴气秘境与潮汐海市。南疆不硬锁低境界玩家进入，但通过推荐境界、敌人强度和危险提示表达风险。

- 巫妖盟：南疆主势力和盟城，包含巫妖盟城门、盟城坊市、巫妖盟悬赏碑、使者馆和巫祝石庭。
- 木灵宗：南疆大型宗门，第一版不允许加入，作为草木宗门地点、商店、采集、对话和试炼入口。
- 百草谷：灵药采集与妖藤战斗区，提供灵草、凝气草和妖丹碎片。
- 万妖山：妖兽战斗与灵宠踪迹地点，提供妖兽骨和妖丹碎片。
- 瘴雾沼泽：瘴毒花采集、蛊虫战斗和沉木机缘地点。
- 千瀑灵崖：秘境区，提供灵瀑守卫战斗和青木灵液机缘。
- 潮汐海市：海边坊市和潮汐传闻地点，售卖南疆材料。
- 归潮礁岛：潮音秘洞与沉珠宝匣秘境，提供潮生贝和灵石奖励。

### 未开放州域

以下州域已在大世界地图中预留，但暂未开放内部地点：

- 东海：万岛沧溟，未来方向为仙舟、海岛、水府、归墟遗迹。
- 西漠：沙海佛国，未来方向为魔宗、佛寺、商道和荒漠秘境。
- 北境：玄冰雪域，未来方向为剑修、冰修、雪原部族和寒域试炼。

新增州域时，应先补充州域主题、主要势力、资源类型、敌人生态和进入条件，再添加具体地点。

## 系统设计

### 修炼与境界

修炼是主成长线。玩家当前通过手动聚气获得修为，宗门状态提供倍率加成。境界突破负责推进版本阶段和系统解锁。

新增境界时必须说明：

- 境界定位和节奏目标。
- 修为需求和突破成本。
- 成功率及失败惩罚。
- 解锁的新系统、内容或地图。

### 历练与地图

历练通过三层结构组织：州域、地点、场景。场景行动包括对话、商店、任务榜、战斗、采集、招募、入宗和机缘。

新增地图内容时必须说明：

- 玩法目的：资源产出、剧情推进、战斗挑战、系统解锁或氛围补充。
- 解锁条件：境界、宗门身份、任务状态、道具或未来剧情条件。
- 玩家收益：修为、灵石、材料、任务进度、队友、声望或系统入口。
- 关联系统：任务、战斗、背包、宗门、洞府或未来系统。

### 战斗

战斗使用小队制回合机制。玩家、同伴、灵宠和敌人拥有气血、灵力、攻击、防御、神识、速度、闪避、暴击等属性。行动顺序按速度排序，伤害技能会先按目标闪避率判定，再根据目标类型和效果类型结算。

当前战斗定位：

- 炼气前期：山狼和守草妖兽提供基础战斗教学。
- 炼气中后期：黑风妖修混合修士和妖兽，测试队伍续航。
- 筑基前后：古修洞府守卫作为更高压力战斗。

### 宗门

青云宗是早期组织线。当前功能包括入宗、身份展示、贡献、声望和任务列表。未来可扩展藏经阁、贡献兑换、师门关系、宗门战和宗门职位。

新增宗门玩法时必须说明：

- 宗门身份如何影响修炼、任务、商店或技能。
- 贡献和声望的获得与消耗。
- 与境界、地图和任务的关系。

### 洞府

洞府当前是筑基后解锁的未来系统入口。计划承载闭关、离线收益、聚灵阵、洞府升级、灵田、炼丹或灵宠培养。

新增洞府玩法时必须说明：

- 是否产生离线收益。
- 收益上限和结算周期。
- 升级消耗、解锁条件和对修炼速度的影响。

### 队友与灵宠

当前队伍最多包含两名成员，已有青羽狐和顾青萝。灵宠偏护主和敏捷，伙伴偏治疗和术法辅助。

未来扩展方向：

- 灵宠成长、进阶、天赋和亲密度。
- 伙伴剧情、好感度、技能学习和队伍定位。
- 队伍容量、站位或战斗策略的进一步设计。

### 视觉与 UI 方向

当前 UI 方向调整为现代玄幻 App：保留修仙题材识别，但减少传统水墨、毛笔、卷轴和纸感表达。界面应更接近暗色手游 HUD，使用深色石墨底、半透明面板、细描边、柔和灵光和紧凑信息层级。

长期规则：

- 主色用于修炼、当前状态和可点击高亮；灵力、神通、奖励、危险等状态使用稳定语义色区分。
- 底部导航、顶部状态、背包装备、战斗槽位、地图标记、宗门与洞府入口优先配合轻量图标增强识别。
- 通用图标使用 `lucide-react`，业务组件通过 `src/components/GameIcon.tsx` 的语义名引用。
- 修仙专属图标，如灵石、丹炉、境界印记、灵宠、宗门令牌和秘境入口，可后续用本地 SVG 或图片补足。
- UI 样式改动若改变交互规则、信息层级或长期视觉规范，需要同步记录在本文档；纯色值微调且不影响规则时可不记录。
- 修炼页的主要成长反馈集中在圆形聚气按钮内，修为进度以球形液面填充展示，独立境界进度条不再常驻。
- 大世界和区域图片地图的标记在缩放时保持稳定视觉尺寸，坐标跟随地图移动，但按钮本身不随地图放大遮挡视野。

## 未来内容方向

- 更高境界：筑基后期之后扩展金丹、元婴等阶段。
- 功法神通：让修炼面板中的功法、神通、强化、洗髓、本命逐步可用。
- 宗门进阶：藏经阁、贡献兑换、职位、师父和宗门战。
- 洞府养成：离线闭关、聚灵阵、灵田、炼丹和洞府升级。
- 大世界扩展：开放东海、西漠、北境的内部地图和专属资源，并继续扩展南疆声望与宗门线。
- 装备体系：让背包中的装备从展示变成属性来源。
- 任务链：从单点任务扩展为跨地区剧情线。

## 新增内容记录模板

新增或改动玩法时，在对应章节补充说明，并按以下格式追加记录：

### 2026-05-18：闪避属性与聚气球形进度
- 玩法目的：让速度类成长更容易理解，并把修炼页最关键的境界进度收进主操作按钮，减少修炼页面板层级。
- 解锁条件：属性与战斗规则全局生效；球形进度在修炼页聚气入口生效；地图标记规则在大世界和南疆区域地图生效。
- 玩家收益：背包详细属性可查看速度、闪避和暴击；伤害技能有概率被完全闪避；聚气按钮会按当前修为进度从底部填满，满进度后可在同一区域尝试突破。
- 关联系统：角色属性、装备加成、旧存档兼容、战斗引擎、修炼面板、大世界地图和区域图片地图。
- UI 规则：玩家可见敏捷属性统一命名为“速度”；大世界/区域地图标记在缩放时保持视觉大小稳定，不随地图放大遮挡画面。
- 影响范围：新增闪避属性和伤害判定，调整修炼页境界进度展示；不改变修为获取、突破成本、地图坐标、任务奖励或商店价格。

### 2026-05-18：现代玄幻 UI 与轻量图标体系
- 玩法目的：把界面从强水墨古风转向更清晰、现代、手游化的玄幻 App 风格，提高移动端识别效率和操作层级。
- 解锁条件：全局 UI 规则，进入主菜单、修炼、背包、历练、战斗、宗门和洞府时均生效。
- 玩家收益：底部导航、顶部状态、修炼、背包、战斗、地图、宗门和洞府入口加入语义图标；主色、灵力色、神通色、奖励色和危险色更清楚。
- 关联系统：`src/styles.css` 主题变量、`lucide-react` 依赖、`src/components/GameIcon.tsx` 图标语义层和各核心 UI 面板。
- UI 规则：主题采用深色石墨底、半透明面板、细描边、柔和灵光和紧凑 HUD；业务组件不直接散用图标组件，优先引用 `GameIcon` 语义名。
- 影响范围：仅影响视觉、图标和信息层级，不改变修为收益、突破公式、战斗属性、奖励产出、商店价格或存档结构。

### 2026-05-18：南疆二级地图开放
- 玩法目的：扩展中州之外的第二个完整州域，让大世界探索从预览进入可游玩的多区域内容。
- 解锁条件：历练随开局解锁，南疆在大世界中显示为已开放；低境界玩家可进入，但地点显示推荐境界和危险提示。
- 玩家收益：可通过南疆二级图片地图进入巫妖盟、木灵宗、百草谷、万妖山、瘴雾沼泽、千瀑灵崖、潮汐海市和归潮礁岛，获得新材料、任务、商店、采集、宝箱和战斗奖励。
- 关联系统：大世界地图、区域图片地图、任务榜、商店、物品、敌人、战斗、背包材料和文档数值。
- UI 规则：南疆使用 `World_map_nanjiang2.png` 作为二级地图，支持拖拽、缩放、重置和地点信息抽屉；中州继续使用卡片式地点地图。
- 影响范围：新增南疆区域内容、区域任务过滤和区域商店过滤；不新增加入木灵宗系统，不改变现有中州地点结构。

### 2026-05-17：战斗 UI 与携带栏重做
- 玩法目的：降低战斗界面噪音，让玩家专注于携带技能、丹药和敌我气血状态。
- 解锁条件：进入任意战斗后生效，玩家仍只控制主角，灵宠、道友和敌人继续自动行动。
- 玩家收益：战斗面板取消当前行动提示框、防御按钮和普攻按钮；顶部右侧并排显示“战斗日志”和小型“逃离”；主操作区只展示两个修士技能、一个神通槽和一个丹药槽。
- 关联系统：战斗引擎、玩家技能、丹药、战斗日志、敌我单位状态。
- UI 规则：气血条使用红色；战斗日志默认收起，点击“战斗日志”弹出浮层，再次点击关闭；日志内主角名显示绿色，灵宠/道友显示蓝色，敌方显示红色。
- 影响范围：新增战斗携带配置与旧存档兼容；不改变技能伤害公式、治疗公式、逃跑概率或敌人 AI 权重。

- 解锁条件：历练和背包随开局解锁，大世界地图为历练第一层，装备页为背包默认页。
- 玩家收益：州域标记更小，地图最高缩放提高到 4 倍，鼠标悬停州域标记不再产生位置跳动；可穿戴装备超过一页时使用上一页/下一页切换。
- 关联系统：大世界地图、州域标记、缩放拖拽、背包装备页、属性详情。
- UI 规则：背包装备页默认只显示气血、攻击、灵力、防御四项核心属性；点击“属性”文字后，在属性区上方弹出详细属性面板，再次点击“属性”关闭弹窗。
- 影响范围：仅影响 UI 交互和信息密度，不改变装备加成、角色属性、战斗公式、物品价格或掉落。

- 玩法目的：背包从单纯展示改为角色装备管理入口，玩家可以在装备页查看穿戴槽、可穿戴装备和最终属性。
