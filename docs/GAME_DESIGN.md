# 游戏设计文档

本文档记录修仙文字 RPG 的长期设计方向。任何新增玩法、内容、地图、系统、解锁或叙事结构时，都必须同步更新本文档。

## 核心体验

玩家从青云城外醒来，以散修身份开始修仙。早期目标是修炼聚气、积累资源、探索中州、加入青云宗、获得灵宠或同伴，并最终突破到筑基阶段。游戏以文字反馈、轻量地图和回合制战斗推进。

当前体验支柱：

- 修炼：通过聚气获得修为，满足境界要求后尝试突破。
- 突破：消耗灵石和材料，按成功率提升境界，并解锁新系统。
- 背包：记录材料、丹药、任务物品和装备展示。
- 历练：通过大世界地图进入州域、地点和场景，触发对话、采集、战斗、任务、入宗和招募。
- 战斗：玩家和最多两名队友对抗妖兽或敌对修士，采用速度排序和回合行动。
- 宗门：加入青云宗后获得身份、贡献、声望和任务联系。
- 洞府：当前开放入口，未来承载闭关、聚灵阵和离线收益。
- 存档：本地三槽位存档，支持设置、继续、新建和删除。

## 当前世界内容

### 中州

中州是早期修行和新手内容中心。

- 青云城：主城，包含城主府、城东大道、城西坊市、城南民居和城北任务榜。
- 天玄城门：中州地图新城门场景，位于 `28,16`，使用竖版城门图展示天玄城正门、守门阵纹与左右两名可对话修士。
- 黑风山：野外战斗区，包含山狼、黑风妖修和妖兽骨产出。
- 灵药谷：采集和战斗区，提供凝气草、灵草，并触发青羽狐事件。
- 古修洞府：秘境区，提供洞府守卫战斗和筑基丹、青木灵液等机缘。
- 落霞镇：任务叙事地点，用于送信任务和商路传闻。

### 南疆

南疆是当前开放的第二个州域，主题为巫妖盟、木灵宗、草木灵药、瘴气秘境与潮汐海市。南疆不硬锁低境界玩家进入，但通过推荐境界、敌人强度和危险提示表达风险。

- 巫妖盟：南疆主势力和盟城，包含巫妖盟城门、盟城坊市、巫妖盟悬赏碑、使者馆和巫祝石庭。
- 木灵宗：南疆大型宗门，第一版不允许加入，作为草木宗门地点、商店、采集、对话和试炼入口。
- 百草谷：灵药采集与妖藤战斗区，提供灵草、凝气草和妖丹碎片。
- 万妖山：妖兽战斗与灵宠踪迹地点，提供妖兽骨和妖丹碎片。
- 瘴雾沼泽：瘴毒花采集、蛊虫战斗和沉木机缘地点。
- 千瀑灵崖：秘境区，提供灵瀑守卫战斗和青木灵液机缘。
- 潮汐海市：海边坊市和潮汐传闻地点，售卖南疆材料。
- 归潮礁岛：潮音秘洞与沉珠宝匣秘境，提供潮生贝和灵石奖励。

### 未开放州域

以下州域已在大世界地图中预留，但暂未开放内部地点：

- 东海：万岛沧溟，未来方向为仙舟、海岛、水府、归墟遗迹。
- 西漠：沙海佛国，未来方向为魔宗、佛寺、商道和荒漠秘境。
- 北境：玄冰雪域，未来方向为剑修、冰修、雪原部族和寒域试炼。

新增州域时，应先补充州域主题、主要势力、资源类型、敌人生态和进入条件，再添加具体地点。

## 系统设计

### 修炼与境界

修炼是主成长线。玩家当前通过手动聚气获得修为，宗门状态提供倍率加成。境界突破负责推进版本阶段和系统解锁。

玩家等级采用“修仙境界等级”，不额外设置传统 `Lv.1 / Lv.2`。每个玩家等级由“大境界 + 小阶段”组成：大境界包括凡人、炼气、筑基、结丹、元婴、化神、炼虚、合体、大乘和飞升后预留；小阶段固定为初期、中期、后期、圆满。当前存档仍保存 `realmId`，但每个境界配置都必须标注 `majorRealmId` 与 `phaseId`，用于修为曲线、装备穿戴要求、区域掉落和敌人难度。

玩家属性采用“存档核心字段 + 运行时派生字段”。存档保存境界、修为、当前气血/灵力、年龄、寿元、心境、悟性、伤势、灵气不稳、灵石和基础战斗属性；境界名、修为上限、战力等显示值从配置和最终属性实时计算。旧存档中的 `power`、`lifespanCurrent/lifespanMax` 和文本心境只作为兼容来源，不再作为真实规则来源。

突破成功后直接套用目标境界基础属性，气血/灵力回满，寿元更新，心境提升并清空灵气不稳。突破失败按小境界和大境界区分惩罚：小境界以修为回落、心境下降和不稳层数为主；大境界额外增加伤势，体现资源门槛。

神识系统在装备 V1 中改为正式受限属性：代码字段统一为 `spiritSense`，元婴前玩家与装备神识默认为 0，元婴阶及以上装备才允许生成神识主属性和神识词条。治疗、回灵、法术命中与反闪避逻辑读取最终神识。

新增境界时必须说明：

- 境界定位和节奏目标。
- 所属大境界与小阶段。
- 修为需求和突破成本。
- 成功率及失败惩罚。
- 解锁的新系统、内容或地图。

### 物品、装备与双轴品级

物品采用双轴设计：器类阶位表示“这件东西属于哪个修仙大境界”，颜色品级表示“这件东西有多稀有、多少词条、视觉反馈有多强”。器类阶位必须压过颜色品级，避免低境界高颜色装备在主属性上压过高境界普通装备。

器类阶位：凡阶、炼气阶、筑基阶、结丹阶、元婴阶、化神阶、炼虚阶、合体阶、大乘阶和飞升后预留。装备、消耗品、材料、符箓和阵盘都可以拥有器类阶位；任务和剧情物品默认不参与普通掉落逻辑。

颜色品级采用 9 档：凡品、良品、精品、灵品、玄品、地品、天品、仙品、神品。颜色品级主要影响颜色、词条数量、特殊效果概率、价格曲线和同阶内的属性浮动，不替代器类阶位。

当前开放规则：

- 每个正式物品配置一个器类阶位和一个当前颜色品级，例如炼气阶凡品回春散、炼气阶良品凝气草、筑基阶灵品筑基丹。
- 装备主属性 V1 改为“境界阶位 × 品级 × 部位换算”：武器读取境界/品级攻击表，其他部位按武器 W 值或专表派生。
- 同大境界内，小阶段影响装备投放：初期以白/绿为主，中期以绿/蓝为主，后期稳定出现蓝/紫，圆满开始投放紫以上、突破材料和 Boss 掉落。
- 紫色灵品以上开始偏用途装备和构筑价值，不做无脑上位替换。
- 装备栏、背包九宫格、物品详情、战斗丹药槽和法宝槽都读取物品品级展示对应 UI 特效。
- 旧测试存档中的旧六品后缀会按兼容映射折回正式物品 ID；旧 `mystic` 映射为新灵品。
- 装备采用单件实例系统：同名装备在背包中作为不同实例保存，实例固化 `name/displayName/realmTier/quality/slot/mainStats/affixes`，穿戴槽保存装备实例 ID。
- 装备栏 V1 为 8 槽：武器、法袍、头冠、护腕、鞋履、戒指、护符和法宝；取消独立神识法宝栏，神识相关词条由法宝或护符承载。旧存档的头冠、鞋履、饰品、法宝槽会迁移到新槽位。

### 历练与地图

历练通过三层结构组织：州域、地点、场景。场景行动包括对话、商店、任务榜、战斗、采集、招募、入宗和机缘。

新增地图内容时必须说明：

- 玩法目的：资源产出、剧情推进、战斗挑战、系统解锁或氛围补充。
- 解锁条件：境界、宗门身份、任务状态、道具或未来剧情条件。
- 玩家收益：修为、灵石、材料、任务进度、队友、声望或系统入口。
- 关联系统：任务、战斗、背包、宗门、洞府或未来系统。

### 战斗

战斗使用小队制回合机制。玩家、同伴、灵宠和敌人拥有气血、灵力、攻击、防御、神识、速度、闪避、暴击和暴击倍率等属性。行动顺序按速度排序；普攻和武器技能吃完整闪避，指向性法术吃 50% 闪避，范围和阵法伤害默认不可闪避。
战斗操作区采用轻量携带栏：第一排显示两个武技槽，第二排显示两个功法槽，第三排显示神通、法宝和丹药。法宝槽读取 `artifact` 装备，拥有主动、领域或法则词条时可催动，否则展示被动生效状态。

当前战斗定位：

- 炼气前期：山狼和守草妖兽提供基础战斗教学。
- 炼气中后期：黑风妖修混合修士和妖兽，测试队伍续航。
- 筑基前后：古修洞府守卫作为更高压力战斗。

### 宗门

青云宗是早期组织线。当前功能包括入宗、身份展示、贡献、声望和任务列表。未来可扩展藏经阁、贡献兑换、师门关系、宗门战和宗门职位。

新增宗门玩法时必须说明：

- 宗门身份如何影响修炼、任务、商店或技能。
- 贡献和声望的获得与消耗。
- 与境界、地图和任务的关系。

### 洞府

洞府是筑基后解锁的长期养成系统。当前已开放闭关与聚灵阵：玩家可以开始闭关，离线或切换页面后回到洞府领取修为；聚灵阵可消耗灵石与材料升级，提高闭关效率和最长闭关时间。灵田、炼丹、灵宠位和洞府禁制先作为后续设施入口展示。

新增洞府玩法时必须说明：

- 是否产生离线收益。
- 收益上限和结算周期。
- 升级消耗、解锁条件和对修炼速度的影响。
- 是否影响存档结构，以及旧存档如何补默认洞府数据。

### 队友与灵宠

当前队伍最多包含两名成员，已有青羽狐和顾青萝。灵宠偏护主和敏捷，伙伴偏治疗和术法辅助。

未来扩展方向：

- 灵宠成长、进阶、天赋和亲密度。
- 伙伴剧情、好感度、技能学习和队伍定位。
- 队伍容量、站位或战斗策略的进一步设计。

### 视觉与 UI 方向

当前 UI 方向调整为现代玄幻 App：保留修仙题材识别，但减少传统水墨、毛笔、卷轴和纸感表达。界面应更接近暗色手游 HUD，使用深色石墨底、半透明面板、细描边、柔和灵光和紧凑信息层级。

长期规则：

- 主色用于修炼、当前状态和可点击高亮；灵力、神通、奖励、危险等状态使用稳定语义色区分。
- 底部导航、顶部状态、背包装备、战斗槽位、地图标记、宗门与洞府入口优先配合轻量图标增强识别。
- 通用图标使用 `lucide-react`，业务组件通过 `src/components/GameIcon.tsx` 的语义名引用。
- 修仙专属图标，如灵石、丹炉、境界印记、灵宠、宗门令牌和秘境入口，可后续用本地 SVG 或图片补足。
- UI 样式改动若改变交互规则、信息层级或长期视觉规范，需要同步记录在本文档；纯色值微调且不影响规则时可不记录。
- 修炼页的主要成长反馈集中在圆形聚气按钮内，修为进度以球形波浪液面填充展示，独立境界进度条不再常驻。
- 修为进度满后，圆形聚气按钮文字改为“突破”；若下一境界需要灵石或材料，点击后弹出突破准备面板，固定显示 5 个材料格，空格只允许放入当前突破配置所需且玩家已拥有的材料堆叠。
- 背包颜色品级展示采用高辨识手游化效果：凡品白灰、良品绿色、精品蓝色、灵品紫色、玄品橙色、地品红色、天品金色、仙品白金、神品赤金；列表保持紧凑，强光效集中在品级徽章、物品卡、装备槽、战斗携带槽和详情浮层，灵品以上物品名称加粗。
- 物品详情浮层只展示玩家决策需要的信息：名称、品级徽章、描述、阶位、类型、数量、核心属性、战斗效果、真实词条、价值和出售；配置曲线、颜色倍率、词条数量上限等策划调参信息不在前台展示。没有实际词条时不显示词条区。装备可用时不额外显示穿戴阶段，境界不足时只在底部主操作按钮提示。
- 背包各分类统一使用 3 列 x 3 行凹槽格子，每页 9 格。装备分类中部标题为“背包”，只显示装备实例，格子内不显示数量；物品、丹药、材料继续显示堆叠数量角标。点击已占格子打开详情，空格只作为占位。
- 已穿装备格只显示部位和装备名，不在格子内展示属性加成或卸下按钮；点击已穿装备后复用装备详情浮层，底部操作为“卸下”和“返回”。
- 有角色热点的场景图应完整展示图片，并在角色附近叠加可点击姓名标记；点击后以弹窗展示 NPC 名称、身份和对话文本。
- 大世界和区域图片地图的标记在缩放时保持稳定视觉尺寸，坐标跟随地图移动，但按钮本身不随地图放大遮挡视野。
- 大世界和区域图片地图的州域/地点入口在普通模式只显示中心格文字标记；文字位置取自 zone `anchor`。大世界州域仍以 `3x3` zone 命中，中州和南疆地点均为 `anchor` 单格入口。

## 未来内容方向

- 更高境界：筑基后期之后扩展金丹、元婴等阶段。
- 功法神通：让修炼面板中的功法、神通、强化、洗髓、本命逐步可用。
- 宗门进阶：藏经阁、贡献兑换、职位、师父和宗门战。
- 洞府养成：离线闭关、聚灵阵、灵田、炼丹和洞府升级。
- 大世界扩展：开放东海、西漠、北境的内部地图和专属资源，并继续扩展南疆声望与宗门线。
- 装备体系：让背包中的装备从展示变成属性来源。
- 任务链：从单点任务扩展为跨地区剧情线。

## 新增内容记录模板

新增或改动玩法时，在对应章节补充说明，并按以下格式追加记录：

### 2026-05-20：装备栏八槽压缩与神识法宝栏取消
- 玩法目的：减少装备区纵向占用，让手机视口内更快看到背包格子，同时把独立神识法宝栏并回法宝构筑。
- 解锁条件：背包随开局解锁；元婴后神识主属性和神识词条仍可在法宝/护符等现有槽位生成。
- 玩家收益：装备格从 9 槽改为 8 槽，单格间距和内边距压缩；已穿装备格只保留部位和装备名，浏览装备页时信息密度更高。
- 关联系统：共享类型、装备槽配置、装备生成、词条槽位、背包 UI、旧存档装备槽标准化和数值文档。
- UI 规则：装备栏标题显示“八槽”，部位为武器、法袍、头冠、护腕、鞋履、戒指、护符和法宝；已穿装备点击后在装备详情浮层中执行“卸下”或“返回”；旧 `spirit_artifact` 装备槽位在标准化时映射为法宝。
- 影响范围：取消独立神识法宝栏，不新增物品、敌人、任务、掉落、商店或奖励。

### 2026-05-20：装备品级、八槽装备与战斗词条 V1
- 玩法目的：把装备从静态属性来源升级为境界阶位、九档品级、八个部位、随机主属性和副词条共同驱动的长期养成系统。
- 解锁条件：背包随开局解锁；神识主属性与神识词条仅元婴阶及以上装备生成；高一大境界装备可试穿但处于封印状态。
